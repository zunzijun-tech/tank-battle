# 坦克大战 (Tank Battle)

经典坦克大战游戏的 HTML/JavaScript 实现，在浏览器中直接运行。

## 玩法

- **目标**：消灭所有敌方坦克，保护基地（右下方的老鹰）
- **敌方坦克**会从上方三个出生点不断出现
- 可以击碎砖墙来开辟道路

## 操作

| 按键 | 功能 |
|------|------|
| ↑ ↓ ← → | 移动坦克 |
| 空格 | 射击 |
| Enter | 开始/重新开始 |
| P | 暂停/继续 |

## 运行

直接用浏览器打开 `坦克大战.html` 即可开始游戏。

## 特性

- 关卡系统（每通关一次进入下一关，地图变化）
- 敌方 AI
- 砖墙（可破坏）和钢墙（不可破坏）
- 爆炸特效
- 老鹰基地保护机制
